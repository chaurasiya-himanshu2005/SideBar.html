# Sidebar-HTML

A clean and responsive sidebar navigation component built using HTML and CSS.

## About

This project demonstrates how to create a structured sidebar layout using semantic HTML and basic CSS styling. It can be reused in dashboards, admin panels, or portfolio websites.

## Built With

- HTML5
- CSS3

## Usage

1. Clone the repository
2. Open `index.html` in your browser

## Purpose

Created for practicing layout design and navigation structure in front-end development.
